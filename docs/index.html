Terminal idéntico a Kali Linux en Ubuntu

Instala Oh My Zsh, prompt exacto de Kali, plugins de autosugerencia y resaltado de sintaxis, y la paleta de colores Kali Dark para Tilix. Un solo comando.

▶ Probar Demo ↓ Instalar
Demo automática
Observa el prompt, el resaltado de sintaxis y el código de error en acción
franvi@kali: ~
Terminal interactiva
Escribe comandos reales. Tab para autocompletar, ↑↓ para historial.
franvi㉿kali — zsh

Haz clic en el terminal y escribe · Tab autocompleta · ↑↓ historial · Ctrl+C cancela · help para ver comandos

¿Qué instala?
8 pasos automatizados, sin intervención manual
⚡

Prompt exacto de Kali

El icónico prompt de dos líneas con el símbolo ㉿, colores diferenciados para root y usuario, y RPROMPT con código de error.

🔌

Plugins zsh

Configuración lista con los plugins más útiles:

  • zsh-autosuggestions
  • zsh-syntax-highlighting
  • git
  • colored-man-pages
  • command-not-found
🎨

Paleta Kali Dark

Colores exactos del terminal de Kali Linux aplicados automáticamente en Tilix vía dconf. JSON standalone incluido para otros terminales.

📝

Aliases estilo Kali

ll, update, please (sudo !!), cls, ip con colores y más.

💾

Historial inteligente

10.000 entradas sin duplicados, compartido entre sesiones con SHARE_HISTORY.

🛡️

Backup automático

El script hace backup de tu .zshrc existente con timestamp antes de reemplazarlo.

Paleta Kali Dark
Haz clic en un color para copiar su hex
Instalación
Un script, todo configurado
git clone https://github.com/Haplee/kali-terminal-setup.git && cd kali-terminal-setup && chmod +x kali-terminal-setup.sh && ./kali-terminal-setup.sh
✔ Ubuntu 20.04+ / Debian 11+
✔ Acceso sudo
✔ Conexión a internet
Tras el script: exec zsh o source ~/.zshrc
